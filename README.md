# Wallet Protect — App Router

Aperçu complet (UX Base) avec logo intégré, App Router, pages:
- `/` (Scan + Approvals Radar)
- `/base` (Protections: Presets, Spend Permissions, Risk Banner)
- `/help` (Aide & Liens)

## Démarrer
```bash
npm install
npm run dev
```

## Déploiement Vercel
- Importer le repo GitHub.
- Node 20, Next 14.
- Variables d’environnement via le dashboard.
- Branch protection et CI (voir `.github/workflows/ci.yml`).
